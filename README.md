# myclaw.rocks

Practical operator-grade website for setting up, operating, and scaling OpenClaw.

## Local preview

From the project root:

```bash
python3 -m http.server 8080
```

Then open: `http://localhost:8080`

## Structure

- `index.html` - home page
- `best-practices.html`
- `architectures.html`
- `resources.html`
- `work-with-us.html`
- `use-cases.html` (scaffold)
- `integrations.html` (scaffold)
- `mistakes.html` (scaffold)
- `assets/css/styles.css` - shared design system
- `assets/js/site.js` - nav/footer + reveal effects
- `resources/templates/*.md` - practical downloadable templates
